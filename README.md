# DockerTools
python里etcd的安装方法:                                                                                                             
安装基础库                                                                                                                           
yum install libffi libffi-devel python-devel                                                                                       
安装程序                                                                                                                            
git clone https://github.com/jplana/python-etcd.git                                                                                
cd python-etcd                                                                                                                     
python setup.py install                                                                                                            
其中python里docker的安装方法是                                                                                                       
easy_install docker-py
